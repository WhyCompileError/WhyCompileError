## WhyCompileError

*"me write code. code no work. why compile error?"*

## The Strategy:
1.  Write code.
2.  Hope the compiler doesn't yell at me.
3.  Wonder why compile error. Cry.
4.  Fix it (hopefully).
5.  Document it so future me suffers less.
